搭建博客

安装hugo

llan@192-168-1-132 ~ % brew install hugo
llan@192-168-1-132 ~ % hugo version
llan@192-168-1-132 ~ % hugo new site myblog
llan@192-168-1-132 ~ % cd myblog/
llan@192-168-1-132 myblog % ls -l
llan@192-168-1-132 myblog % pwd
/Users/llan/myblog

下载模版并新建文章

llan@192-168-1-132 myblog % git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c
#启动hugo
llan@192-168-1-132 myblog % hugo server -t m10c --buildDrafts
#创建文章
llan@192-168-1-132 myblog % hugo new post/firstblog.md

启动hugo

进入myblog根目录下，再启动

llan@192-168-1-132 post % cd ../..
llan@192-168-1-132 myblog % hugo server -t m10c --buildDrafts

本地启动：http://localhost:1313/

部署

  1. Repository name: 与用户名一致且小写
    例如: angaoanzj.github.io

  2. 生成public文件
